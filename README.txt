

Project1Main.py contains the program that generates the triples for project 1 and saves them in "output/TheTriples.txt". 


Project2Main.py contains the university chatbot for Project 2.


An important note is that to run this program, rdflib, isodate beautifulSoup, urllib, Pandas, pycurl are the libraries that need to be installed for project 1. To run the program for project 2, the aiml_bot library must be installed.
